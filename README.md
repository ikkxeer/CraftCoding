# CraftCoding (DEVELOPED WITH THE HELP OF AI 😇 )
Application developed to promote the non-use of AI in programming, a tool to focus on and avoid using AI during code development.

## Where does the inspiration for the program come from?
During my second year in an advanced vocational training program in cross-platform application development, I started noticing a worrying trend: many students were relying entirely on AI to write their code, often without fully understanding it.

While AI is undoubtedly a powerful tool, this growing dependence risks weakening fundamental programming skills and reducing long-term cognitive development.

This realization led me to start developing an idea focused on addressing this problem.

## Tech stack
 
- **UI** — [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter)
- **Window scanning** — [PyGetWindow](https://github.com/asweigart/pygetwindow)
- **Process monitoring** — [psutil](https://github.com/giampaolo/psutil)
- **Certificate generation** — [Pillow](https://pillow.readthedocs.io/en/stable/)
- **Database** — SQLite3 (stdlib)
- **Keyboard/mouse automation** — [PyAutoGUI](https://github.com/asweigart/pyautogui)
